# 📡 JJY シミュレーター

電波時計が電波を受信できない環境で、時刻合わせを補助するWebツールです。

🔗 **サイト**: https://yukimodeki.github.io/jjy-simulator/

---

## 📖 JJY電波とは？

**JJY** は、日本標準時（JST）を送信する長波標準電波です。

| 送信所 | 周波数 | 場所 |
|--------|--------|------|
| おおたかどや山標準電波送信所 | 40 kHz | 福島県 |
| はがね山標準電波送信所 | 60 kHz | 佐賀県 |

電波時計はこのJJY電波を受信して、自動的に正確な時刻に合わせます。
しかし、**鉄筋コンクリートの建物の中・地下・電子機器の近く**では電波が届きにくいことがあります。

---

## 💡 このツールの仕組み

本ツールは **Web Audio API** を使って、JJY電波と同じ時刻符号をパソコンやスマートフォンのスピーカーから再生します。

スピーカーや音声回路が発生する**微弱な電磁ノイズ**を、電波時計のフェライトアンテナが拾うことで、時刻合わせができる場合があります。

> ⚠ **注意**: 実際のJJY電波（40/60 kHz）はスピーカーの可聴域を超えているため、完全な再現ではありません。効果は端末・時計の機種・環境によって異なります。

---

## 🕐 JJY信号の構造

JJYは1分間に60ビットの時刻情報を送信します。毎秒1ビット、振幅変調（AM）で符号化されています。

### 振幅変調の仕組み

```
1秒間のイメージ：

通常       ████████████████████  (振幅100%)
          ↓ 秒の頭で振幅が下がる
マーカー   ██░░░░░░░░░░░░░░░░░░  (0.2秒間 低振幅 → マーカービット)
1ビット    ████░░░░░░░░░░░░░░░░  (0.5秒間 低振幅 → 1)
0ビット    ████████░░░░░░░░░░░░  (0.8秒間 低振幅 → 0)
```

### 60ビットのデータ構造

| 秒 (ビット位置) | 内容 |
|----------------|------|
| 0 | マーカー M |
| 1〜3 | 分の十の位 (BCD: 40, 20, 10) |
| 4 | 未使用 |
| 5〜8 | 分の一の位 (BCD: 8, 4, 2, 1) |
| 9 | マーカー P1 |
| 10〜11 | 未使用 |
| 12〜13 | 時の十の位 (BCD: 20, 10) |
| 14 | 未使用 |
| 15〜18 | 時の一の位 (BCD: 8, 4, 2, 1) |
| 19 | マーカー P2 |
| 20〜21 | 未使用 |
| 22〜23 | 通算日の百の位 (BCD: 200, 100) |
| 24 | 未使用 |
| 25〜28 | 通算日の十の位 (BCD: 80, 40, 20, 10) |
| 29 | マーカー P3 |
| 30〜33 | 通算日の一の位 (BCD: 8, 4, 2, 1) |
| 34〜35 | 未使用 |
| 36 | パリティビット PA1（時のEvenパリティ） |
| 37 | パリティビット PA2（分のEvenパリティ） |
| 38 | 未使用 |
| 39 | マーカー P4 |
| 40〜41 | 未使用 |
| 42〜45 | 年の十の位 (BCD: 8, 4, 2, 1) |
| 46〜48 | 年の一の位 上3桁 (BCD: 8, 4, 2) |
| 49 | マーカー P5 |
| 50 | 年の一の位 下1桁 (BCD: 1) |
| 51〜53 | 曜日 (BCD: 4, 2, 1 / 0=日〜6=土) |
| 54〜55 | うるう秒予告 (LS1, LS2) |
| 56〜58 | 未使用 |
| 59 | マーカー P0 |

## 🚀 使い方

### 1. サイトを開く
https://yukimodeki.github.io/jjy-simulator/ をブラウザで開きます。

### 2. 音量を最大にする
スマートフォンやPCの**音量を最大**にしてください。
音量が小さいと電磁ノイズが弱くなり、受信できません。

### 3. 周波数を選ぶ
デフォルトは **10 kHz（推奨）** です。
受信できない場合は他の周波数を試してください。

| 周波数 | 特徴 |
|--------|------|
| 1 kHz | 音が聞こえやすい（低周波） |
| 3 kHz | 低周波寄りのバランス型 |
| 5 kHz | バランス型 |
| **10 kHz** | **推奨。多くの環境で有効** |
| 15〜20 kHz | ほぼ聞こえない（高周波） |

### 4. 「▶ 開始」を押す

### 5. 端末を時計に密着させる
電波時計の**裏面や側面**にあるフェライトアンテナ部分に
端末をできるだけ近づけてください（1〜2 cm以内が理想）。

```
  スマホ
 ┌──────┐
 │      │
 │      │  ← ここを
 └──────┘
      ↕ 密着（1〜2cm）
 ┌──────────┐
 │ 電波時計 │
 └──────────┘
```

### 6. 数分間待つ
電波時計が受信モードに切り替わるまで、**2〜5分**かかることがあります。
時計によっては受信ボタンを押すと強制受信モードになります。

## 🔧 受信できない場合のチェックリスト

- [ ] 音量が最大になっているか
- [ ] 端末を時計のアンテナ部分に密着させているか
- [ ] 時計の向きを変えてみたか（フェライトアンテナは指向性あり）
- [ ] 周波数を変えて試したか（1〜20 kHz）
- [ ] テレビ・PC・蛍光灯など電子機器から離したか
- [ ] 深夜に試したか（電波環境が安定しやすい）

## 🛠 技術仕様

- **使用技術**: HTML / CSS / JavaScript（Web Audio API）
- **対応ブラウザ**: Chrome, Firefox, Safari, Edge（最新版）
- **対応端末**: PC・スマートフォン・タブレット
- **時刻基準**: [timeapi.io](https://timeapi.io) 経由でNTP同期（毎秒取得・RTT補正済み）
- **符号化方式**: JJY時刻符号（BCD + マーカー + パリティ）
- **音声生成**: `OscillatorNode` + `GainNode` による振幅変調
- **フォント**: Noto Sans JP / 游ゴシック
